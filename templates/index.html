Kings League World Cup 2024 API

Overview

Welcome to the documentation for the Players API. Below are the available endpoints:

  • GET /players: Retrieve all players.
  • POST /player: Create a new player.
  • POST /player/<id>: Update a player by ID.
  • DELETE /player/<id>: Delete a player by ID.

GET /players

Retrieve all players.

GET /players

            "name": "Player Name",

            "yellow_cards": 2,
            "red_cards": 0,
            "mvps": 1,

            "ratio": 0.5

        {
            "id": 2,

Example Usage

curl -X GET https://api.example.com/players

POST /player

Create a new player.

Request Body

{
    "name": "New Player",
    "role": "Midfielder",
    "team": "Team A",
    "appearances": 5,
    "goals": 2,
    "yellow_cards": 1,
    "red_cards": 0,
    "mvps": 0,
    "goals_conceded": 0,
    "ratio": 0.4
}

Response

{
    "message": "Player created successfully",
    "player": {
        "id": 3,
        "name": "New Player",
        "role": "Midfielder",
        "team": "Team A",
        "appearances": 5,
        "goals": 2,
        "yellow_cards": 1,
        "red_cards": 0,
        "mvps": 0,
        "goals_conceded": 0,
        "ratio": 0.4
    }
}

Example Usage

curl -X POST -H "Content-Type: application/json" -d '{"name": "New Player", "role": "Midfielder", "team": "Team A", "appearances": 5, "goals": 2, "yellow_cards": 1, "red_cards": 0, "mvps": 0, "goals_conceded": 0, "ratio": 0.4}' https://api.example.com/player

POST /player/<id>

Update a player by ID.

Request Body

{
    "name": "Updated Player Name",
    "role": "Forward",
    "team": "Team B",
    "appearances": 20,
    "goals": 15,
    "yellow_cards": 2,
    "red_cards": 0,
    "mvps": 3,
    "goals_conceded": 10,
    "ratio": 0.75
}

Response

{
    "message": "Player updated successfully",
    "player": {
        "id": 1,
        "name": "Updated Player Name",
        "role": "Forward",
        "team": "Team B",
        "appearances": 20,
        "goals": 15,
        "yellow_cards": 2,
        "red_cards": 0,
        "mvps": 3,
        "goals_conceded": 10,
        "ratio": 0.75
    }
}

Example Usage

curl -X POST -H "Content-Type: application/json" -d '{"name": "Updated Player Name", "role": "Forward", "team": "Team B", "appearances": 20, "goals": 15, "yellow_cards": 2, "red_cards": 0, "mvps": 3, "goals_conceded": 10, "ratio": 0.75}' https://api.example.com/player/1

DELETE /player/<id>

Delete a player by ID.

Request

DELETE /player/1

Response

{
    "message": "Player deleted successfully",
    "deleted_player": {
        "id": 1,
        "name": "Updated Player Name",
        "role": "Forward",
        "team": "Team B",
        "appearances": 20,
        "goals": 15,
        "yellow_cards": 2,
        "red_cards": 0,
        "mvps": 3,
        "goals_conceded": 10,
        "ratio": 0.75
    }
}

Example Usage

curl -X DELETE https://api.example.com/player/1
